# Day 01 — Project Start

## Project Title
ESP32 Smart Dual Temperature Monitor with IoT Logging

## Project Aim
To build an ESP32-based system that reads air temperature, humidity, and probe temperature, displays the readings on an OLED screen, triggers buzzer alerts, and uploads data to an IoT platform.

## Components Available
- ESP32
- DHT11 sensor
- DS18B20 sensor
- OLED display
- Buzzer
- Breadboard and wires

## Planned Features

### Part 1
- Read DHT11 (temp + humidity)
- Read DS18B20 (probe temp)
- Display readings on OLED
- Trigger buzzer alerts
- Send data to IoT dashboard

### Part 2
- Phone control (change thresholds)
- Turn alerts on/off from phone
- View live data on phone
